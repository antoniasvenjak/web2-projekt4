4. projekt - HTTP2

Napredni razvoj programske potpore za web

HTTP/2 je objavljen 2015. godine kao RFC 7540 specifikacija s ciljem otklanjanja nedostataka HTTP/1.1 protokola. Glavna razlika u odnosu na stariju verziju je način grupiranja i prijenosa podataka između klijenta i poslužitelja.
HTTP/2 donosi značajna poboljšanja performansi kroz multipleksiranje, kompresiju i server push, zadržavajući pri tome poznatu HTTP semantiku i povećavajući sigurnost web komunikacije.
Cilj ove demonstracije je prikazati da se web stranica s pripadajućim resursima brže učitava korištenjem HTTP/2 protokola. Ova stranica sadrži 15 slika, 5 jednostavnih CSS datoteka i 5 jednostavnih JS datoteka.
